Датасет `fake_job_postings.csv` содержит объявления о вакансиях (название, текст описания, требования, компания и т.д.)  
Целевой признак — столбец `fraudulent`:

- `0` — объявление выглядит реальным
- `1` — объявление помечено как мошенническое (фейк)

**Цель:** по признакам вакансии (текст + метаданные) предсказать, является ли вакансия фейковой.  
Это задача **бинарной классификации**.
